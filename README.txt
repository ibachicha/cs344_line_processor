TO COMPLILE PROGRAM:

gcc --std=gnu99 -o line_processor -g -Wall line_processor.c -pthread

TO RUN PROGRAM:

./line_processor

TO RUN WITH INPUT REDIRECTION:

./line_processor < INPUTFILE_NAME

TO RUN WITH OUTPUT REDIRECTION:

./line_processor > OUTPUTFILE_NAME

TO RUN WITH INPUT AND OUTPUT REDIRECTION:

./line_processor < INPUTFILE_NAME > OUTPUTFILE_NAME
